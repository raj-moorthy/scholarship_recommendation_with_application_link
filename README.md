# 🎓 Scholarship & Gender Prediction App

This is a Streamlit-based web application that predicts:
1. **User Gender** based on their full name.
2. **Scholarship Recommendation** based on user-provided description (education, income, caste, etc.).

The system leverages a deep learning model trained on scholarship data and uses a pre-trained gender detection module.

---

## ✅ Features

- ✅ Predict gender from user name using `gender-guesser`.
- ✅ Recommend scholarships dynamically based on user input description.
- ✅ Display official scholarship link for easy access.
